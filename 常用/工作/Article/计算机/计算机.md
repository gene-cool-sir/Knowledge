二进制（Binary）：二进制是计算机中最基本的数字系统，只包含0和1。它是由二进制位（bit）组成的，每个位可以是0或1。了解二进制对于理解计算机内部的工作原理至关重要。
        二进制 0 对应十进制 0
        二进制 1 对应十进制 1
        含义：只包含两个数字，0和1。
        对应关系：每一位代表2的幂次方，从右向左递增。例如，1010（二进制）表示 1 * 2^3 + 0 * 2^2 + 1 * 2^1 + 0 * 2^0 = 10（十进制）。
        基本类型字节数：通常用于表示计算机的最小存储单元，例如，一个字节（8位）。

十进制（Decimal）：十进制是我们日常生活中最常用的数字系统，包含0到9的数字。十进制使用权重为10的幂次方来表示数字的位置，例如，数值153表示1 * 10^2 + 5 * 10^1 + 3 * 10^0。
        十进制 0 对应二进制 0000
        十进制 1 对应二进制 0001
        十进制 2 对应二进制 0010
        十进制 9 对应二进制 1001
        含义：包含十个数字，0到9。
        对应关系：每一位代表10的幂次方。例如，123（十进制）表示 1 * 10^2 + 2 * 10^1 + 3 * 10^0 = 123（十进制）。
        基本类型字节数：通常使用不同的数据类型，如整数（int）和浮点数（float）来表示，字节数取决于具体实现和平台。

十六进制（Hexadecimal）：十六进制是一种较为常见的表示数字的方式，它使用0到9的数字和字母A到F（表示10到15）来表示数字。十六进制可以更紧凑地表示二进制数据，并经常在计算机科学和编程中使用。
        十六进制 0 对应二进制 0000
        十六进制 1 对应二进制 0001
        十六进制 2 对应二进制 0010
        ...
        十六进制 F 对应二进制 1111
        含义：包含十个数字和六个字母，0-9 和 A-F，表示十六进制数。
        对应关系：每一位代表16的幂次方。例如，1A（十六进制）表示 1 * 16^1 + 10 * 16^0 = 26（十进制）。
        基本类型字节数：常用于表示内存地址和二进制数据，通常字节数的一半，例如，一个字（4位）通常由两个十六进制数字表示。
        

八进制（Octal）：八进制使用0到7的数字来表示数值。尽管在计算机科学中使用较少，但在某些场景下仍然有一定的用途。
        八进制 0 对应二进制 000
        八进制 1 对应二进制 001
        八进制 2 对应二进制 010
        八进制 7 对应二进制 111
        含义：包含八个数字，0到7。
        对应关系：每一位代表8的幂次方。例如，64（八进制）表示 6 * 8^1 + 4 * 8^0 = 52（十进制）。

### 在Java中数据的序列化与反序列化, 写出对象属性与读入属性

```java
/**
 * 数据按照一定格式写出,按照一定格式读入
 */
public class DataInputStreamTest {

    public static void main(String[] args) throws IOException {
        Person person = new Person();
        person.setAge(20);
        person.setUsername("gene");
        ByteArrayOutputStream bos = new ByteArrayOutputStream();
        ObjectOutputStream stream = new ObjectOutputStream(bos);
        // 写入2个字节,并写入name的字符数据
        stream.writeUTF(person.getUsername());
        // 写入4个字节,并写入name的字符数据
        stream.writeInt(person.getAge());
        byte[] bytes = bos.toByteArray();
        InputStream inputStream = new ByteArrayInputStream(bytes);
        DataInputStream dataInputStream = new DataInputStream(inputStream);
        // 按照写入的顺序读取字符数据,它首先从输入流中读取两个字节，这两个字节表示了后续字符串的长度。然后，它根据长度读取相应数量的字节，并将这些字节解码为字符串。在示例代码中，name变量将接收读取到的字符串数据。
        String i1 = dataInputStream.readUTF();
        // 用于读取整数数据。它从输入流中读取4个字节，并将这些字节解码为一个整数值
        int i = dataInputStream.readInt();

        System.out.println(i + i1 );
    }
}
```
